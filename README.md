# Student Mental Health Analysis

## Overview
This project analyzes student mental health survey data to identify patterns affecting wellbeing and academic pressure.

## Tools Used
- SQL
- Excel
- Power BI

## Objectives
- Clean and structure survey data
- Analyze trends in student mental health
- Identify relationships between academic factors and wellbeing

## Key Insights
- Students reporting high academic pressure showed higher stress levels
- Sleep patterns were strongly correlated with reported mental wellbeing

## Project Structure
dataset.csv – sample dataset  
queries.sql – SQL queries used for analysis  
dashboard.png – visualization of findings
